# IT Analytics Projects

This repository contains hands-on IT and data analytics projects built using Python.
Each project focuses on solving practical problems using simple, explainable logic.

Projects

 1. IT Asset Performance Dashboard
Analyzes IT infrastructure data to compute Health and Risk scores.
The project helps identify high-risk production systems that require maintenance priority.

**Key concepts used:**
- Resource health analysis (CPU, memory, disk)
- Operational risk scoring
- Prioritization of production-critical assets

Project folder: IT_Asset_Performance_Dashboard
